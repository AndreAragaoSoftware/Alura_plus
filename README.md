# Como executar o projeto


## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/AndreAragaoSoftware/Alura_plus

# entrar na pasta do projeto Alura plus
cd Alura_plus

# instalar dependências
yarn install

# executar o projeto
yarn start
```
